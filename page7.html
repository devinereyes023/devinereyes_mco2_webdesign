Exercise Reflection

Based on the video that I watched, I now understand how inline, internal, and external CSS work with HTML. Inline style are easy to add right to each element but are hard to update if you have a lots of them. Internal CSS keeps all styling for one page together at the top, which is better for small sites, though copying it to other pages gets messy. Enternal CSS saves styles in a separate file linked to HTML changing just that one file updates every page, making it neat and simple to manage once you get yhe hang on it.
